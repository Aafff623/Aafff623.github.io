## 6. git 

常用 Git 命令

初始化仓库
```
git init
```

添加远程仓库
```
git remote add origin <仓库地址>
```

添加全部文件到暂存区
```
git add .
```

提交到本地仓库
```
git commit -m "提交说明"
```

推送到远程仓库
```
git push origin master
```

拉取远程仓库内容
```
git pull origin master
```

查看提交历史
```
git log
```

回退到最近一次提交
```
git reset --hard HEAD
```
